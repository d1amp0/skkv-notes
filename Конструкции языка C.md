## if

`if` выполняет ветвление по условию.

```c
if (x > 0) {
    printf("positive\n");
} else if (x < 0) {
    printf("negative\n");
} else {
    printf("zero\n");
}
```

Условие интерпретируется по правилам C:

- `0` — ложь;
- любое ненулевое значение — истина.
--------------------------------------------------------------------------------
## Циклы: while, do while, for

### `while`

Проверяет условие до входа в тело цикла.

```c
while (x > 0) {
    --x;
}
```

### `do while`

Тело выполняется хотя бы один раз, а условие проверяется после него.

```c
do {
    read_next();
} while (need_more());
```

### `for`

Компактная запись цикла с инициализацией, условием и шагом.

```c
for (int i = 0; i < n; ++i) {
    printf("%i\n", i);
}
```

## `break`, `continue`, `return`

- `break` завершает ближайший цикл или `switch`;
- `continue` переходит к следующей итерации цикла;
- `return` завершает функцию.
--------------------------------------------------------------------------------
## switch

`switch` удобен, когда нужно выбрать одно из нескольких действий по целочисленному или перечислимому значению.

```c
switch (x) {
case 1:
    printf("one\n");
    break;
case 2:
    printf("two\n");
    break;
default:
    printf("other\n");
}
```

Что важно помнить:

- без `break` выполнение "проваливается" в следующий `case`;
- `default` необязателен, но полезен для полноты логики;
- `switch` часто работает вместе с `enum`.

На лекции также упоминался `Duff's device` — необычный прием, использующий проваливание `case` внутри цикла для ручного разворачивания.

## goto и метки

`goto` делает безусловный переход к метке внутри той же функции.

```c
if (error) {
    goto cleanup;
}

cleanup:
free(ptr);
```

Хотя `goto` часто считают "грязной" конструкцией, в системном C он бывает удобен для единообразной очистки ресурсов при ошибках.
