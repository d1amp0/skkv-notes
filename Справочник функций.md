В этом разделе собираются не только функции, но и некоторые фундаментальные инструменты языка, которые в практике часто вспоминают рядом со стандартной библиотекой.

## Анатомия функции и прототипы

Программа на C компилируется пофайлово. Если функция вызывается до своей реализации или находится в другом файле, компилятору нужен прототип.

- **Прототип** — объявление вида `int f(void);`.
- **Реализация** — полное определение с телом в `{}`.
- В современных стандартах C вызов функции без корректного объявления недопустим.

## `printf`

- Библиотека: `<stdio.h>`.
- Назначение: форматированный вывод.

```c
printf("sum = %i\n", a + b);
```
--------------------------------------------------------------------------------
## `fprintf`

- Библиотека: `<stdio.h>`.
- Назначение: форматированный вывод в файл.

```c
fprintf(file, "sum = %i\n", a + b);
```
--------------------------------------------------------------------------------
## `scanf`

- Библиотека: `<stdio.h>`.
- Назначение: форматированный ввод.
- Возвращает количество успешно прочитанных объектов.
- Для записи в переменную надо передавать указатель на неё (см. [[Указатели и массивы#Что такое указатель|указатели]]).

```c
int x;
if (scanf("%i", &x) == 1) {
    printf("%i\n", x);
}
```
--------------------------------------------------------------------------------
## `sscanf`

- Библиотека: `<stdio.h>`.
- Назначение: форматированный ввод из строки.
- Возвращает количество успешно прочитанных объектов.
- Для записи в переменную надо передавать указатель на неё (см. [[Указатели и массивы#Что такое указатель|указатели]]).

```c
int x;
if (sscanf(buffer, "%i", &x) == 1) {
    printf("%i\n", x);
}
```

## `fscanf`

- Библиотека: `<stdio.h>`.
- Назначение: форматированный ввод из файла.
- Возвращает количество успешно прочитанных объектов.
- Для записи в переменную надо передавать указатель на неё (см. [[Указатели и массивы#Что такое указатель|указатели]]).

```c

    printf("%i\n", x);

```
--------------------------------------------------------------------------------
## `fread`

- Библиотека: `<stdio.h>`.
- Назначение: чтение блока байтов из потока.
- Удобна для двоичных форматов и массивов объектов.

Сигнатура:

```c
size_t fread(void *ptr, size_t size, size_t count, FILE *stream);
```

Смысл параметров:

- `ptr` — куда писать данные;
- `size` — размер одного элемента;
- `count` — сколько элементов читать;
- `stream` — поток.

Возвращает количество **успешно прочитанных элементов**, а не число байтов.

--------------------------------------------------------------------------------
## `fwrite`

- Библиотека: `<stdio.h>`.
- Назначение: запись блока байтов в поток.

Сигнатура:

```c
size_t fwrite(const void *ptr, size_t size, size_t count, FILE *stream);
```

Возвращает количество успешно записанных элементов.

Важно:

- `fread` и `fwrite` не решают автоматически вопросы endian-ности, выравнивания и переносимости бинарного формата;
- запись "сырой" структуры в файл удобна, но привязывает формат к ABI и платформе.

## `sizeof`

`sizeof` — это не функция, а оператор языка.

- Возвращает размер объекта или типа в байтах.
- Имеет тип результата `size_t`.
- Для массива в той же области видимости возвращает полный размер массива, а не размер указателя.

```c
int arr[10];
size_t bytes = sizeof(arr);
size_t count = sizeof(arr) / sizeof(arr[0]);
```
--------------------------------------------------------------------------------
## `strlen

- Библиотека: `<string.h>`.
- Назначение: длина C-строки без завершающего `'\0'`.

Сигнатура:

```c
size_t strlen(const char *s);
```

Пример:

```c
strlen("abc") == 3
```
--------------------------------------------------------------------------------
## `strcmp`

- Библиотека: `<string.h>`.
- Назначение: лексикографическое сравнение строк.

Сигнатура:

```c
int strcmp(const char *a, const char *b);
```

Результат:

- `0`, если строки равны;
- `< 0`, если `a` меньше `b`;
- `> 0`, если `a` больше `b`.

Это удобно, потому что условие `strcmp(a, b) <= 0` по смыслу соответствует `a <= b` в лексикографическом порядке.
